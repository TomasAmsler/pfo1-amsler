# Práctica Formativa Obligatoria 1 - Portafolio Personal

Este proyecto consiste en una Landing Page de portafolio personal desarrollada como parte de la Tecnicatura en Desarrollo de Software (IFTS N.°29). El objetivo es demostrar habilidades en maquetación con HTML5 semántico y estilización con CSS3, utilizando diseño responsivo mediante FlexBox y tipografías externas como Google Fonts.

- **¿Qué ventajas encontraste al utilizar Flexbox en tu proyecto?**
  *Respuesta:* Flexbox facilitó enormemente la alineación de las tarjetas de proyectos y películas, permitiendo que se distribuyan de forma equilibrada y se adapten automáticamente al ancho de la pantalla sin romper el diseño.

- **¿Qué animación o transición implementaste y por qué?**
